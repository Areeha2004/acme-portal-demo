# Acme Portal (demo app)

A small React + Vite sample app (login → dashboard) used to demo the AI build/preview
→ confirm → PR → staging → production flow.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build   # outputs to dist/
```

Deploys to Cloudflare Pages: build command `npm run build`, output directory `dist`.
